## توضیحات پروژه 📌
<!-- Project Description : استاندارد -->
این پروژه یک رابط کاربری خلاقانه و تعالی از نوع **Rotating Navigation** است که در آن با کلیک روی دکمه منو، بخش اصلی صفحه چرخش پیدا می‌کند و منوی ناوبری به شکل متفاوتی نمایش داده می‌شود.

هدف اصلی این پروژه، تمرین طراحی رابط‌های کاربری مدرن و کار با افکت‌های حرکتی در فرانت‌اند است.

در این پروژه تمرکز روی مفاهیم زیر بوده است:

- ساخت منوی ناوبری خلاقانه
- استفاده از CSS Transform و Transition
- کنترل کلاس‌ها و تعاملات با JavaScript
- طراحی رابط کاربری تعاملی

## Project Description 📌
<!-- Project Description : standard -->
This project is a creative and interactive Rotating Navigation UI where clicking the menu button rotates the main content area and reveals the navigation in a unique way.

The main goal of this project is to practice modern UI design and motion effects in frontend development.

This project focuses on:

Building a creative navigation menu
Using CSS transforms and transitions
Controlling interactions with JavaScript
Designing an interactive user interface

---


## Features ✨
<!-- Features : standard -->
- Unique rotating navigation menu

- Modern and engaging user interface
